Detailed Description

template<uint32_t GPIO_BASE, uint16_t PIN, uint8_t ADC_CHANNEL>
class Potentiometer< GPIO_BASE, PIN, ADC_CHANNEL >

Templated static class that allows to initialize and get the state of an ADC1 channel from a pin at any given time In order to choose a pin, please refer to the datasheet, searching for ADC1 available pins, and choosing the right ADC_CHANNEL related to them

Template Parameters
GPIO_BASEGPIO base for ADC pin
PINGPIO pin to be used as ADC input
ADC_CHANNELADC channel related to the pin (See Datasheet)
